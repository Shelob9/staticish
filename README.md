# WP API To Static

Converts WordPress posts and pages to static JSON and markdown.


## Usage

### Install

```
yarn install...
```

### Use

```
const wpToStatic = require( 'wp-to-static' );

....
```

### Develop

- Clone
    - `git clone git@github.com:Shelob9/wp-api-to-static.git`
- Install
    - `cd wp-api-to-static && yarn`
- Test
    - `yarn test`
- Build
    - `yarn build`
